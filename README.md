## Steps to Run the Project

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Grechir/CookBook.git
   cd CookBook

**Set up the backend:**
```
  cd project
  pip install -r requirements.txt
  python manage.py runserver
```
   
**Set up the frontend (in a new terminal):**
```
  cd ../frontend
  npm install
  npm start
```
